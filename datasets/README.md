# CyperTeam NetOpt Datasets

## Overview

This directory contains datasets used for research and experimentation in the **CyperTeam NetOpt** project.

The purpose of these datasets is to support the development and evaluation of AI-driven optimization techniques for large-scale IoT networks.

The datasets may include network telemetry data, traffic analysis data, and experimental networking results.

---

# Dataset Categories

The datasets used in this project may fall into several categories.

## Network Telemetry Data

This dataset contains performance metrics collected from network infrastructure.

Typical metrics include:

- network latency
- packet loss
- throughput
- bandwidth utilization
- network congestion indicators

These datasets help analyze network behavior and identify performance bottlenecks.

---

## IoT Device Communication Data

This dataset represents communication patterns between IoT devices and network gateways.

Examples of collected information may include:

- device transmission intervals
- packet transmission statistics
- device connectivity status
- network routing behavior

These datasets help study traffic patterns in distributed IoT systems.

---

## Traffic Analysis Data

Traffic analysis datasets may include information related to:

- network traffic volume
- packet flow distribution
- communication patterns between network nodes

These datasets are useful for training machine learning models capable of identifying optimal network configurations.

---

## Experimental Network Data

Datasets generated from experimental networking environments.

These datasets may include:

- results from simulated network experiments
- telemetry data from wireless network testbeds
- edge computing performance metrics

Experimental datasets help evaluate optimization algorithms under realistic network conditions.

---

# Dataset Format

Datasets may be stored in different formats depending on the experiment.

Common formats include:

- CSV
- JSON
- Parquet
- structured telemetry logs

Each dataset should include documentation describing its structure and fields.

Each dataset should include the following information:

- dataset description

# Data Usage

The datasets contained in this directory are intended for research and experimental purposes.

Possible applications include:

- training machine learning models
